# ReactJS Custom Hook - useFetch

## Project Overview

I made this project to understand how custom hooks work in React. I created a reusable custom hook called useFetch which fetches data from an API and displays it in a photo gallery.

## Features

- Custom Hook (useFetch)
- API Data Fetching
- Loading State
- Error Handling
- Photo Gallery Layout

## Challenges Faced

While creating this project, I faced some difficulty understanding how useEffect works with API calls. I also learned how to manage loading and error states properly.

## What I Learned

- Creating custom hooks in React
- Using useState and useEffect
- Fetching API data using fetch()
- Writing reusable code

- ReactJS
